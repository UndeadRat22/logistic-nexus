# Logistic Nexus Sprite Layout

Both runtime sprite sheets contain 32 frames arranged as 8 columns by 4 rows.

## Main Sheet

- Runtime file: `logistic-nexus.png`
- Expanded template: `assembling-machine-1.png`
- Frame size: 278 x 290 pixels
- Original machine area: x 0-213, y 64-289
- Reserved top area: 64 pixels
- Reserved right area: 64 pixels

## Shadow Sheet

- Runtime file: `logistic-nexus-shadow.png`
- Expanded template: `assembling-machine-1-shadow.png`
- Frame size: 264 x 247 pixels
- Original shadow area: x 0-189, y 82-246
- Reserved top area: 82 pixels
- Reserved right area: 74 pixels

At the prototype scale of 0.5, each 64-pixel reserved strip represents one
32-pixel Factorio tile. Add matching artwork separately to every frame.

The unmodified source sheets are retained as `assembling-machine-1-original.png`
and `assembling-machine-1-shadow-original.png`.
